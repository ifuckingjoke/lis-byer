⚠️ Warning: lis-byer does not collect information about your browsers, cookies, or sessions. The script uses pre-installed browsers for Playwright, automatically creating a new session upon startup.

🦊 Lis-Byer is a script for automating the filling of XLSX tables related to purchasing skins in games such as CS2, DOTA2, and RUST. It uses the popular and verified skin market [Lis-Skins](https://lis-skins.com/ru/blog/) as its data source.

## ⚙️ How to work?

* Download the current release: [Release-v2.0.1](https://github.com/ifuckingjoke/lis-byer/releases)

* Run the executable file lis-byer.exe. After that, navigate to the newly created data directory and add skin links to the data.txt file. 
    * *Alternatively:* You can manually create the necessary directories before running the EXE: books for generated tables and data/data.txt for your links.

* Launch the lis-byer.exe file again.

After analyzing all links, the script will generate a table:

![Example](example.png)

This script is useful for traders, bulk skin purchases, various audits and reports, or creating custom projects using XLSX tables.
